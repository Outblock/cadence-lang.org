Cadence is designed for AI-native development. Three integrations make Cadence documentation and language intelligence available to any AI agent:

- **[Skills](./skills)** — inject Cadence knowledge into AI coding agents with one command
- **[MCP Server](./mcp-server)** — give AI agents live access to documentation search and the Cadence language server
- **[LLM Endpoints](./llm-endpoints)** — plain-text documentation endpoints optimized for LLM context windows

## Agent integrations

Step-by-step setup guides for specific AI tools:

- [Antigravity](./integrations/antigravity) — skill and MCP setup
- [Claude](./integrations/claude) — Claude Code, Claude Desktop, Claude API
- [Cursor](./integrations/cursor) — project rules and MCP setup
- [OpenAI Codex](./integrations/codex) — AGENTS.md and context injection
- [Gemini](./integrations/gemini) — Gemini CLI and API
- [OpenCode](./integrations/opencode) — MCP configuration
